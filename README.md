# NOIRFEATHER · GitHub Governance

This `.github` directory contains the shared governance, communication, and contribution standards for the NOIRFEATHER holding.

## Holding Structure

NOIRFEATHER operates through three legal entities:

- `NOIRFEATHER L.L.C.` (United States): official documentation in English.
- `NOIRFEATHER S.A.S` (Colombia): official documentation in Spanish.
- `NOIRFEATHER OÜ` (Estonia, EU): official documentation in English and Estonian.

## Documentation Language Policy

English is the primary and canonical language for cross-entity documentation, platform standards, and global communication.
Localized versions are maintained according to legal and regional requirements.

## Directory Scope

- `profile/README.md`: organization-level profile visible on GitHub.
- Shared templates and policy files for issues, pull requests, security, and governance.

## Governance Objectives

- Consistent intake and triage for issues.
- Clear pull request expectations and review quality.
- Reliable communication of organizational standards.
- Security-first maintenance practices.

## Maintenance Standards

- Keep policy files aligned with actual operations.
- Update legal, compliance, and security contacts when ownership changes.
- Prioritize precise, operational language over ambiguous wording.
- Preserve terminology consistency across English, Spanish, and Estonian versions.
